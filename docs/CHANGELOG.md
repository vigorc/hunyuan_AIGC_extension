# Changelog

## [0.1.0] - 2024

### Added
- Initial release of AIGC Extension
- Image upload functionality (single images or folders)
- 3D model generation via Hunyuan API
- Deformable physics support with customizable parameters
- Rigid body physics with multiple collider types
- Auto-load models to stage option
- Keep GLB files option (prevent auto-deletion after loading)
- **Scale Tool**: Interactive scaling feature for generated models
  - Select prim in viewport to get dimensions
  - Input actual height to automatically calculate scale factor
  - Support for X, Y, or Z axis as height axis
  - Uniform scaling to match real-world dimensions
  - Detailed scaling information output

### Fixed
- Extension loading issues in Omniverse
- Missing documentation files
- Dependency declarations in extension.toml
